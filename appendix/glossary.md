# Glossary

This glossary defines all the technical terms used in this book. Terms are organized alphabetically.

---

## A

**APT (Advanced Package Tool):** The command-line tool for installing, updating, and removing software on Ubuntu. Example: `sudo apt install firefox`

**Application (App):** A program you use to do something. Examples: Firefox (web browser), Calculator, LibreOffice (word processor).

**Application Menu:** The menu showing all installed programs. Click the grid of 9 dots to open it.

**Arch Linux:** An advanced Linux distribution that you build yourself. Not for beginners, but great for learning.

---

## B

**Bash:** The default shell (command interpreter) in Ubuntu's Terminal. Runs the commands you type.

**BIOS (Basic Input/Output System):** Low-level software that starts when you turn on your computer, before the operating system loads.

**Bluetooth:** Wireless technology for connecting devices like headphones, mice, and keyboards to your computer.

**Bookmark:** A saved link to a website in your browser, for easy access later.

**Boot:** The process of starting up your computer. "Booting" means turning it on and loading the operating system.

**Browser:** A program for viewing websites on the Internet. Examples: Firefox, Chrome.

**Bug:** An error or problem in software that makes it not work correctly.

---

## C

**Cache:** Temporary storage of data for faster access. Browsers cache web pages; the system caches software packages.

**Case-Sensitive:** When uppercase and lowercase letters are treated differently. Linux filenames are case-sensitive: `File.txt` is different from `file.txt`.

**Command:** An instruction you type in the Terminal to make the computer do something. Example: `ls` lists files.

**Command Line:** Another name for the Terminal. The text-based way to control your computer.

**CPU (Central Processing Unit):** The "brain" of your computer that does all the calculations and processing. Measured in GHz (gigahertz).

**Creative Commons (CC):** A type of license that lets people share and use creative work freely, with certain conditions. This book uses CC BY-NC-SA.

---

## D

**Debian:** A community-driven Linux distribution. Ubuntu is based on Debian.

**Desktop:** The main screen you see after logging in, where you can put files and see your wallpaper.

**Desktop Environment:** The graphical interface that makes Ubuntu look and work the way it does. Ubuntu uses GNOME by default.

**Directory:** Another word for folder. A container that holds files and other folders.

**Distribution (Distro):** A packaged version of Linux with specific software and settings. Examples: Ubuntu, Linux Mint, Fedora.

**Dock:** The bar on the left side of Ubuntu's screen showing your favorite apps and currently running programs.

**Download:** To copy a file from the Internet to your computer.

**Dual Boot:** Having two operating systems installed on one computer. You choose which to start when you turn on the computer.

---

## E

**Email:** Electronic mail. Messages sent over the Internet from one person to another.

**Encrypt:** To scramble data so only authorized people can read it. Important for security.

**Extension:** The ending of a filename that shows what type of file it is. Examples: `.txt` (text), `.jpg` (image), `.mp3` (audio).

---

## F

**File:** A collection of data stored with a name. Can be a document, picture, video, program, etc.

**Filesystem:** The way your computer organizes and stores files and folders.

**Firefox:** A free, open-source web browser that comes pre-installed on Ubuntu.

**Folder:** A container that holds files and other folders. Also called a directory.

**Format:** To erase everything on a disk and prepare it for new data. **Warning:** This deletes all files!

**Free Software:** Software that respects users' freedom to run, study, modify, and share it. Different from "free" meaning zero cost.

---

## G

**GB (Gigabyte):** A unit of data size. 1 GB = 1,000 MB = 1,000,000 KB = 1,000,000,000 bytes. A movie is usually 1-5 GB.

**GHz (Gigahertz):** A measure of CPU speed. One GHz = one billion cycles per second. Higher = faster processing.

**GIMP (GNU Image Manipulation Program):** Free, open-source image editing software, like Photoshop.

**GitHub:** A website where people share open-source code and collaborate on projects.

**GNOME:** The desktop environment used by default in Ubuntu. Provides the interface, windows, and menus.

**GNU:** A collection of free software tools that work with the Linux kernel. Technically, Ubuntu is "GNU/Linux."

**GPL (General Public License):** A free software license that ensures software stays free forever. If you modify GPL software, you must share your changes.

**GUI (Graphical User Interface):** Pronounced "gooey." The visual way to use a computer with windows, icons, and menus. Opposite of command line.

---

## H

**Hard Drive (HDD):** Traditional storage device with spinning disks. Slower than SSD, but cheaper per GB.

**Hardware:** The physical parts of a computer you can touch: keyboard, mouse, screen, CPU, etc.

**Home Folder:** Your personal folder containing all your files. Usually located at `/home/yourusername`. The `~` symbol represents your home folder.

**HTTPS:** Secure version of HTTP. The "S" means data is encrypted. Always look for `https://` in web addresses when entering passwords.

---

## I

**Icon:** A small picture representing a program, file, or folder.

**Input:** Data or commands you give to the computer (keyboard typing, mouse clicks, etc.).

**Install:** To set up a program on your computer so you can use it.

**Internet:** A global network of connected computers sharing information.

**IP Address:** A unique number identifying a device on a network. Like a street address for computers.

**ISO File:** A single file containing an entire operating system, ready to be burned to USB or DVD for installation.

---

## K

**KB (Kilobyte):** A unit of data size. 1 KB = 1,000 bytes. A page of text is about 2 KB.

**Kernel:** The core of an operating system. In Ubuntu, this is the Linux kernel, which communicates directly with hardware.

**Keyboard Shortcut:** A key combination that performs an action quickly. Example: Ctrl + C copies text.

---

## L

**LibreOffice:** Free, open-source office suite (like Microsoft Office). Includes Writer, Calc, Impress, and more.

**License:** Legal terms that define how software can be used, modified, and shared.

**Linux:** The kernel (core) of Ubuntu and many other operating systems. Created by Linus Torvalds in 1991.

**LTS (Long Term Support):** Ubuntu versions supported with updates for 5 years. Released every 2 years (April of even-numbered years).

---

## M

**Malware:** Malicious software designed to harm your computer or steal data. Includes viruses, spyware, ransomware.

**MB (Megabyte):** A unit of data size. 1 MB = 1,000 KB. A photo is usually 1-5 MB.

**Menu:** A list of options or commands. The Applications Menu shows all your programs.

**Mount:** To make a storage device (USB drive, hard disk) available for use in the filesystem.

---

## N

**Network:** Computers connected together to share information. The Internet is the biggest network.

**Notification:** A message that pops up to inform you about something (new email, update available, etc.).

---

## O

**Open Source:** Software whose source code is publicly available for anyone to view, modify, and share.

**Operating System (OS):** The main software that makes a computer work and lets you run other programs. Examples: Ubuntu, Windows, macOS.

**Output:** Information the computer gives you (images on screen, sounds from speakers, etc.).

---

## P

**Package:** Softwarethat's been packaged for easy installation. Ubuntu uses .deb packages.

**Package Manager:** Software that installs, updates, and removes other software. Ubuntu uses APT.

**Partition:** A section of a hard drive treated as a separate disk. Useful for dual booting.

**Password:** A secret word or phrase that proves who you are. Keep it secret and make it strong!

**Path:** The location of a file or folder. Example: `/home/username/Documents/story.txt`

**Permission:** Rules about who can read, write, or run a file. Important for security.

**Plug-in (or Plugin):** Small software that adds features to a larger program. Example: browser plugins.

**Port:** A connection point for plugging in devices (USB port) or a number for network services.

**Process:** A running program. One program can have multiple processes.

**Prompt:** In Terminal, the text showing you're ready to type a command. Looks like: `username@computer:~$`

**Proprietary:** Software whose source code is secret and controlled by a company. Opposite of open source.

---

## R

**RAM (Random Access Memory):** Your computer's short-term memory. Stores data for programs currently running. Measured in GB.

**Repository (Repo):** A collection of software packages available for installation. Ubuntu has official repositories with thousands of programs.

**Resolution:** The number of pixels on your screen. Higher resolution = sharper image. Example: 1920x1080 (Full HD).

**Root:** The superuser account with complete control over the system. Be careful when using root powers!

**Router:** A device that connects your computer to the Internet and manages your home network.

---

## S

**Screenshot:** A picture of what's on your screen. Press PrtScn to take one.

**Search Engine:** A website that helps you find things on the Internet. Examples: Google, DuckDuckGo, Bing.

**Server:** A computer that provides services to other computers. Web servers host websites; email servers handle email.

**Shell:** A program that interprets commands in the Terminal. Bash is Ubuntu's default shell.

**Shortcut:** See Keyboard Shortcut.

**Software:** Programs and applications that run on your computer. Opposite of hardware.

**Software Center:** Ubuntu's app store for finding, installing, and managing programs.

**Source Code:** Human-readable instructions that programmers write. Computers convert it to machine code to run.

**SSD (Solid State Drive):** Modern storage device with no moving parts. Much faster than traditional hard drives.

**Storage:** Where your computer saves files permanently. Hard drives and SSDs are storage devices.

**sudo:** "Superuser do" - a command that gives you temporary admin powers. Use carefully!

**Suspend:** Sleep mode. Computer uses almost no power but wakes up quickly.

---

## T

**Tab:** Multiple web pages open in one browser window. Each page is a tab.

**Terminal:** A text-based way to control your computer by typing commands. Also called command line or shell.

**Touchpad:** The flat surface below a laptop keyboard that works like a mouse.

**Trash:** Where deleted files go before being permanently removed. Like Windows Recycle Bin.

---

## U

**Ubuntu:** A user-friendly Linux distribution based on Debian. The operating system you're learning about!

**Uninstall:** To remove a program from your computer. Also called "remove."

**Update:** A new version of software with improvements and bug fixes.

**Upgrade:** Installing a newer version of software or the entire operating system.

**Upload:** To send a file from your computer to the Internet.

**URL (Uniform Resource Locator):** A web address. Example: `https://ubuntu.com`

**USB (Universal Serial Bus):** A standard for connecting devices (keyboard, mouse, storage) to computers.

**User:** A person who has an account on the computer. Each user has their own files and settings.

**Username:** The name you use to log in. Also appears in the Terminal prompt.

---

## V

**Virtual Machine (VM):** A computer simulated by software. Lets you run one operating system inside another.

**Virus:** Malicious software that spreads to other computers and causes harm. Rare on Linux.

---

## W

**Wallpaper:** The background picture on your desktop.

**Web Browser:** See Browser.

**Website:** A collection of related web pages on the Internet.

**Wi-Fi:** Wireless networking technology. Lets your computer connect to the Internet without cables.

**Window:** A rectangular area on screen showing a program's interface. Can be moved, resized, minimized, or closed.

**Windows:** A proprietary operating system made by Microsoft. Not to be confused with "windows" (multiple program windows).

---

## X

**X11 (X Window System):** The old display system used by Linux. Being replaced by Wayland.

---

## Z

**Zip File:** A compressed file that contains one or more files, saving space. Needs to be extracted to use the files inside.

---

## Symbols

**~ (Tilde):** Represents your home folder in paths. `~/Documents` means `/home/yourusername/Documents`

**/ (Forward Slash):** Separates folders in a path. Example: `/home/user/Documents`

**. (Dot):** Current directory in Terminal. `./program` runs a program in the current folder.

**.. (Two Dots):** Parent directory. `cd ..` moves up one folder level.

**$ (Dollar Sign):** Shown in Terminal prompt. Means you're a regular user (not root).

**# (Hash):** Shown in Terminal prompt when you're root (superuser). Be very careful!

---

## Quick Reference: Common Commands

**Navigation:**
- `pwd` - show current location
- `ls` - list files
- `cd` - change directory

**Files:**
- `touch` - create empty file
- `cat` - view file contents
- `cp` - copy file
- `mv` - move/rename file
- `rm` - delete file

**System:**
- `sudo` - run as superuser
- `apt` - package manager
- `reboot` - restart computer
- `clear` - clear Terminal screen

---

**Remember:** If you see an unfamiliar term, come back to this glossary!
