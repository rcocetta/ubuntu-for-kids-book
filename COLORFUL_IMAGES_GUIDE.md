## How to Download and Add Images

### From Wikimedia Commons:

1. **Find the image page**
2. **Check the license** (look for CC BY, CC BY-SA, or CC0)
3. **Click "More details"** or scroll down
4. **Right-click the image** and "Save Image As"
5. **Download highest resolution** available
6. **Note attribution requirements**

### Proper Attribution Format:

For CC BY or CC BY-SA images:
```markdown
*Image: [Title] by [Author], [License], via [Source]*
```

Example:
```markdown
*Image: Basic Desktop Computer System by Wikimedia user, CC BY 3.0, via Wikimedia Commons*
```

### Where to Save:

```
images/
├── screenshots/     ← Ubuntu screenshots
├── illustrations/   ← Diagrams (already has 6 SVGs)
└── photos/         ← Create this for CC hardware photos
```

## Color Palette for Consistency

To keep images visually cohesive, prefer images with these colors:

**Ubuntu Brand Colors:**
- Orange: #E95420 (main brand color)
- White: #FFFFFF
- Black: #2C001E (dark aubergine)

**GNOME/Desktop Colors:**
- Purple: #300A24 (terminal background)
- Blue: #4A90E2 (links, selections)
- Green: #8AE234 (terminal text)

**Educational Colors:**
- Bright, primary colors for kids
- High contrast for readability
- Avoid muddy or dull colors
