# Screenshot Guide for "Ubuntu for Kids" ðŸ“¸

This document provides detailed instructions for capturing every screenshot needed for the book. Follow these instructions to ensure consistency and quality across all images.

## General Screenshot Guidelines ðŸ“‹

### Setup Before You Start:
1. **Use a clean Ubuntu 24.04 LTS installation** (or the target version)
2. **Use default theme** - Keep Ubuntu's default appearance settings
3. **Screen resolution**: 1920x1080 recommended for clarity
4. **Close unnecessary windows** - Only show what's needed for each screenshot
5. **Screenshot tool**: Use GNOME Screenshot (press `PrtSc` key) or Flameshot for annotations

### File Naming Convention:
- Save as PNG format (best quality for screenshots)
- Name format: `chapter##-description.png`
- Example: `chapter02-applications-menu.png`
- Store in: `/images/screenshots/`

### Quality Checklist:
- [ ] No personal information visible (names, emails, accounts)
- [ ] Clear, readable text (no blurriness)
- [ ] Appropriate window size (not too small)
- [ ] Clean desktop (no distracting background items)
- [ ] Consistent theme/appearance across all screenshots
