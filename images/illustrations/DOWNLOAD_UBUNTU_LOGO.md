# Download the Ubuntu Logo

The Ubuntu logo should be placed here for the introduction page.

## How to Download:

**Option 1: Direct Download**
```bash
cd images/illustrations
wget https://upload.wikimedia.org/wikipedia/commons/9/9e/Ubuntu_logo_copyleft_1.svg -O ubuntu-logo.svg
```

**Option 2: Manual Download**
1. Go to: https://commons.wikimedia.org/wiki/File:Ubuntu_logo_copyleft_1.svg
2. Click on the image
3. Right-click and "Save As"
4. Save as `ubuntu-logo.svg` in this folder

**Option 3: From Ubuntu Design**
1. Visit: https://design.ubuntu.com/downloads
2. Download official assets
3. Use the SVG version
